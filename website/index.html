Install Jarvis

Two install options per platform. All links below are valid and included in this project.

Linux

Use curl for one-command install or clone with git.

Download install_linux.sh Clone with git 
bash install_linux.sh
Jarvis

Windows

Run PowerShell installer script or clone with git and install locally.

Download install_windows.ps1 Clone with git 
powershell -ExecutionPolicy Bypass -File install_windows.ps1
.\Jarvis.bat
